# FREELANCERAMIT.IN

## Project Structure

- `src/App.js` – Main app, routing
- `src/components/Sidebar.js` – Sidebar navigation
- `src/components/TopServices.js` – Top services for homepage
- `src/data/services.js` – List of all services
- `src/pages/Home.js` – Homepage
- `src/pages/Services.js` – Services grid
- `src/pages/ServiceDetails.js` – Service details
- Other pages: PriceDetails, Documents, ContactUs, About, Login, Signup

## How to run

1. Install dependencies: `npm install`
2. Start the app: `npm start`
3. Add your profile photo: `public/images/your-photo.jpg`
4. Update your degree, qualifications, and details in `Home.js`
5. Update contact info in `ContactUs.js`

## How to add/change services

Just edit `src/data/services.js` – no need to change any other file!
